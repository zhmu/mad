  MAD inventory file format 1.0
  Done at 25 August 1999 by RiNK
---------------------------------

This file consists of this 64-byte header:
旼컴컴컴컫컴컴컴컴컴컴컴쩡컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴커
 Offset  Size (bytes)  Description                                        
쳐컴컴컴컵컴컴컴컴컴컴컴탠컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴캑
      0             4  ID code 0x21564e49 (INV!)                          
      4             2  Version number (100 = 1.0, 201 = 2.01, etc.)       
      6             2  Number of inventory items in the file              
      8            56  Reserved                                           
읕컴컴컴컨컴컴컴컴컴컴컴좔컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴켸

Every inventory item consists of this 128-byte structure:
旼컴컴컴컫컴컴컴컴컴컴컴쩡컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴커
 Offset  Size (bytes)  Description                                        
쳐컴컴컴컵컴컴컴컴컴컴컴탠컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴캑
      0            16  Item name, NULL terminated                         
     16             4  Maximum quantity                                   
     20            16  Inventory icon sprite file (NULL terminated)       
     36             4  Flags                                              
     40            88  Reserved                                           
읕컴컴컴컨컴컴컴컴컴컴컴좔컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴켸

Flags are bits defined as:
旼컴컴쩡컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴커
 Bit  Description                                                          
쳐컴컴탠컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴캑
  0   Undroppable item                                                     
읕컴컴좔컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴켸

The rest are reserved bits and should be zero.
