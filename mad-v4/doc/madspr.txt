 Offset  Size (bytes)  Description

      0             4  ID code 0x21525053 (SPR!)                          
      4             2  Version number (100 = 1.0, 201 = 2.01, etc.)       
      6             2  Sprite height                                      
      8             2  Sprite width                                       
     10             4  Flags                                              
     14             3  Transperancy color pair                            
     15             1  Unused                                             
읕컴컴컴컨컴컴컴컴컴컴컴좔컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴켸
